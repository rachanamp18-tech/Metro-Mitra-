# 🚇 Metro Mitra

Metro Mitra is a smart journey companion I'm building for Bengaluru's Namma Metro — it helps estimate crowd levels, plan journeys, and figure out the right exit gate so you're not lost on the platform.

Built using HTML, CSS, and Vanilla JavaScript. No frameworks, no backend yet — just trying to get the core idea working first.

## Features

- Journey planner between metro stations
- Crowd prediction based on time and day
- Support for Purple, Green, and Yellow lines
- Automatic interchange detection
- Estimated travel time and number of stops
- Suggestions for the best time to travel
- Station-wise exit gate info
- Coach and exit guidance
- Light and dark theme
- Responsive design (mobile, tablet, desktop)

## Technologies Used

## Quick Note - Crowd Data

The crowd levels you see right now are simulated, not real BMRCL data — there's no public live feed available yet, so I built a pattern-based estimate for now. If real data ever becomes accessible, the plan is to swap this out for the real thing.
