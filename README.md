# 🚇 Metro Mitra

Metro Mitra is a smart journey companion for Bengaluru's Namma Metro. It helps users estimate crowd levels, plan journeys, and find the appropriate exit gates for their destination stations.

This project is built using HTML, CSS, and Vanilla JavaScript without any external frameworks.

## Features

- Journey planner between metro stations
- Estimated crowd prediction based on time and day
- Support for Purple, Green, and Yellow metro lines
- Automatic interchange detection
- Estimated travel time and number of stops
- Best travel time suggestions
- Station-wise exit gate information
- Coach and exit guidance
- Light and Dark theme
- Responsive design for mobile, tablet, and desktop

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
